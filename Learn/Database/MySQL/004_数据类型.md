# MySQL数据类型


### 整形

* TINYINT

有符号值：-2^7到2^7-1
无服务值：0到2^8-1

* SMALLINT

有符号值：-2^15到2^15-1
无服务值：0到2^16-1

* MEDIUMINT

有符号值：-2^23到2^23-1
无服务值：0到2^24-1

* INT

有符号值：-2^15到2^15-1
无服务值：0到2^32-1

* BIGINT

有符号值：-2^15到2^15-1
无服务值：0到2^64-1


### 浮点型

* FLOAT

* DOUBLE


### 日期类型

* YEAR

* TIME

* DATE

* DATETIME

* TIMESTAMP


### 字符型

* CHAR(M)

M字节，0 <= M <= 255

* VARCHAR（长度可变）

* TINYTEXT

* TEXT（最大长度为64K变长文本）

* MEDIUMTEXT（）

* LONGTEXT
