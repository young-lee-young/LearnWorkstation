# 路由器命令


### 模式命令

* 三种模式

用户模式: Router>
特权模式: Router#
全局模式: Router(config)#

* 模式切换

```sh
# 进入特权模式
enable

# 退出特权模式
disable

# 进入全局模式
configure terminal

# 退出全局模式
exit

# 退出控制台
quit
```


### 基本命令

* 查看路由器版本

```sh
show version
```

* 配置主机名

```sh
# 全局模式下

hostname 主机名
```

* 设置密码

```sh
# 全局模式下

# 设置明文密码
enable password 密码

# 设置密文密码，两个都设置，只有密文生效
enable secret 密码
```

* 设置时间

```sh
# 进入特权模式

# 设置时间
clock set 10:10:10 31 jul 2011

# 设置时区
clock timezone Beijing +8

# 显示时间
show clock
```

### 基本配置

* 设置路由器允许telnet

```sh
# 全局模式

# 同时允许多少个连接
line vty 0 连接数

# 设置telnet密码
password 密码

# 必须输入密码
login
```

* 配置域名解析

```sh
# 全局模式

ip domain-lookup
ip name-server 8.8.8.8
```

* 配置文件

```sh
# 特权模式

# 查看正在运行的配置
show running-config

# 保存配置
copy running-config startup-config

# 清空配置文件
erase startup-config
reload
```

* 配置以太网接口

```sh
# 启动接口模式
interface ethernet 0/1

# 配置地址
ip address 地址 子网掩码

# 启动接口
no shutdown
```

* 配置广域网接口

设置时钟震荡频率一端DCE，被动接受时钟震荡频率的一端DTE

```sh
interface serial 0/1

# 设置时钟震荡频率（控制网速）
clock rate 64000

ip address 地址 子网掩码

no shutdown

# 查看时钟震荡频率及类型
show controllers serial 1/0
```
