# 条件转移指令



注意⚠️：所有条件转移指令都是段内转移（-128 - 127）



### 零条件转移指令：jz(jump if zero)、jnz(jump if not zero)

```assembly
; 计算 1 + 2 + 3 + 。。。 + 99 + 100
mov ax, 0
mov cx, 100
sum:
		add ax, cx
		dec. cx
		jz end	; 如果 cx 为0，跳转到 end 处
 		jmp sum ; 否则继续执行
		
end:
		nop
```
