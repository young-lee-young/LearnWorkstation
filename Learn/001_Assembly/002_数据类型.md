

* 定义数据

```assembly
; declare byte
db 100

; declare word
dw 0xaa55

; declare double word
dd 0x12345678
```



* 二进制表示

```assembly
; 0b 表示二进制
db 0b00001111

; b 表示2进制
db 11110000b

; 可以使用下划线分开
db 0b0000_1111
db 1111_0000b
```



* 十六进制表示

```assembly
; 0x 表示16进制
db 0xaa55

; h 表示16进制
dw 55aah
```



* 字符串

```assembly
; 字符串
db "hello world!!!", 0, 12, 13
```
