
### echo命令

* 显示变量

```
read your_name		# 接受标准输入
echo "${your_name}, hello!"
```

* 开启转义

```
echo -e "hello! \n"		# -e开启转义，-n换行
echo "lee"

echo -e "hello! \c" 	# -c不换行
```

* 将结果定向到文件

```
echo "hello world!" > test.txt
```

* 原样输出，使用单引号

```
echo '$name\"'			# 输出结果是$name\"
```

* 显示命令执行结果

```
echo `date`				# 反单引号
```
