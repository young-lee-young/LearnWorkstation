# Linux基础学习


### 端口基础知识

* 端口号知识

熟知端口号：IANA把这些端口号指派给了TCP/IP体系中最重要的一些协议
0 - 1023            # 公认端口
登记端口号：为没有熟知端口号的应用程序使用，使用这类端口号必须在IANA按照规定的手续登记，以防止重复
1024 - 49151        # 注册端口
短暂端口号：留给客户进程选择短暂使用
49152 - 65535       # 私有端口


### 注意⚠️

端口号只具有本地意义，只是为了标识计算机应用层的各个进程


### 端口号查看

```sh
lsof -i:端口号

netstat -tunlp | grep 端口号
```


### 路由器查看
