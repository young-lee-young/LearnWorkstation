### slice

* 切片结构体

```go
type slice struct {
	array unsafe.Pointer
	len   int
	cap   int
}
```

* 示例
          | ----- Len ----- |
Data ---> | Var | Var | Var | Var | ... | Var |
          | --------------- Cap ------------- |

* 切片创建

1. 根据数组创建
2. 字面量：编译时插入创建数组的代码
3. make：运行时创建数组

* 扩容机制

1. 期望容量大于当前容量2倍，就会使用期望容量
2. 当前切片长度小于 1024，将容量翻倍
3. 当前切片长度大于 1024，每次增加 25%
4. 切片扩容时，并发不安全，注意切片并发要加锁
