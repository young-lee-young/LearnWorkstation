### Unicode

变长格式

1. 128 个 US-ASCII 字符只需 1 个字节编码
2. 西方常用字符需要 2 个字节
3. 其他字符需要 3 个字节
4. 极少数需要 4 个字节


### 字符串大小

str 的大小为 16 字节，指针 str 8 字节 + 长度 len 8 字节

```go
package main

import (
	"fmt"
	"unsafe"
)

func main() {
	str := "李lee"
    fmt.Println(unsafe.Sizeof(str))
}
```


### 字符串结构体

```go
package runtime

import (
	"unsafe"
)

type stringStruct struct {
	str unsafe.Pointer
	len int // Byte数组的长度，不是字符个数
}
```

* 示例

           | ------- 李 ------- |
Data --->  | Byte | Byte | Byte | Byte | ... | Byte |
           | ---------------- Len ----------------- |


### 字符串访问

* 访问每个字符

```go
package main

import (
	"fmt"
)

func main() {
	s := "李lee"
	for _, char := range s {
		fmt.Printf("%c\n", char)
	}
}
```
