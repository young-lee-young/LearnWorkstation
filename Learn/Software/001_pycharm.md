# Pycharm使用快捷键总结


### 代码编辑方面

1. 折叠代码: CRTL + '-'
2. 展开代码: CRTL + '+'        
3. Ctrl + Alt + L    格式化代码
4. Ctrl + /          注释行
5. Ctrl + D          复制行
6. Ctrl + Y          删除行
7. Shift + Enter     另起一行
8. Home              光标移动到行首
9. Ctrl + Home       光标移动到最开始
10. End              光标移动到行尾
11. Ctrl + End       光标移动到最结尾


### 代码调试

1. Debug下一行: F8
2. 表达式求值: ALT + 'F8' 运行表达式: CTRL + 'ENTER'


### 版本控制

1. Ctrl + K            commit
2. Ctrl + Shift + k    push
