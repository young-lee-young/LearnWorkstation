### 框架

* guest 中的驱动

virtio-net：虚拟网络接口

virtio-blk：块设备驱动

virtio-scsi：SCSI 设备驱动

virtio-ballon：内存气球驱动

virtio-rng：随机数生成器驱动，提供硬件随机数生成

* virtio transport（即 frontend，用来 virtio 信息传输）

virtio-pci

virtio-mmio

virtio-channel-io

* backend（qemu/hypervisor 处理 virtio 信息）


### virtqueues 数据结构

* buffer 区域

buffer 数据区：存储 virtio 传递的数据

* 描述符表

描述符表是一个列表

```c
// 描述符
struct virtq_desc {
    _virtio64 addr;   // buffer 地址
    _virtio32 len;    // buffer 长度
    _virtio16 flags;  // 
    _virtio16 next;   // 指向下一个描述符，因为一次读写操作会有很多数据，存储到很多 buffer 中
}
```

* 可用环形队列

待处理的描述符，由 guest 写入
 
```c
// 可用环形队列
struct virtq_avail {
    _virtio16 flags;
    _virtio16 idx;
    _virtio16 ring[ /* Queue size */ ]; // 存放的也是描述符列表的索引号
    _virtio used_event;
}
```

* 已用环形队列

已经处理的描述符，由 host 写入

```c
struct virtq_used_elem {
    _virtio32 id;  // 描述符列表的索引
    _virtio32 len; // 写入到 buffer 中的数据长度
}

// 已用环形队列
struct virtq_used {
    _virtio16 flags;
    _virtio16 idx;
    struct virtq_used_elem_ring[ /* Queue Size */ ]
}
```
