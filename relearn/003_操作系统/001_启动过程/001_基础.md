### 操作系统启动过程

1. 开机加电，此时 CPU 处于实模式
2. 此时 CS=0xffff，IP=0x0000，CPU 寻址 0xffff（ROM BIOS 的映射区）
3. BIOS 硬件自检
4. BIOS 读取 0 磁头 0 磁道第 1 个扇区（MBR，主引导扇区，512 Byte），如果是 0x55aa 结尾，说明这个扇区是可引导的，将扇区内容加载到内存 0x7c00 处
5. 跳转到 0x7c00 处开始执行 BootLoader
