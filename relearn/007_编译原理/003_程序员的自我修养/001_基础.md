# 程序编译链接过程

预编译 -> 编译 -> 汇编 -> 链接

### 实验源码

```c
#include<stdio.h>

int main() {
    printf("Hello World\n");
	return 0;
}
```

### 编译过程

* 预编译

```bash
# -E 表示预编译
gcc -E hello.c -o hello.i
```

* 编译：编译是整个编译链接过程的重点

编译后生成的是汇编语言

编译过程：词法分析 -> 语法分析 -> 语义分析 -> 中间语言生成 -> 目标代码生成和优化

```bash
# -S 表示只进行编译，不进行汇编
gcc -S hello.i -o hello.s
```

* 汇编

汇编将汇编语言转换成机器指令

```bash
# .o 文件为目标文件（object file）
as hello.s -o hello.o
```

* 链接

```bash
ld
```
