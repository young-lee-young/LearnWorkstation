# 配置网络


### 导入网络配置

网络默认使用NAT模式

网络配置文件位置/usr/share/libvirt/networks/default.xml

```bash
virsh net-define $net_name.xml
```


### 删除网络配置

```bash
virsh net-undefine $net_name
```


### 启动网络

```bash
virsh net-start $net_name
```


### 查看网络

```bash
# 查看启用网络
virsh net-list

# 查看所有网络
virsh net-list --all
```


### 关闭网络

```bash
virsh net-destroy $net_name
```
