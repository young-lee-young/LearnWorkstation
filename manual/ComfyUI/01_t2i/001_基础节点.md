### Load Checkpoint

* 作用

加载模型文件

* 输出

model：模型

clip（Contrastive Language-Image Pre-Training）： 与模型配套的文本编码器，用于将文本转化为向量

vae（Variational Auto-Encoder）：解码器，用于将潜在的图像解码为可见的 RGB 图像

* 使用

Add Node -> loaders -> Load Checkpoint


### CLIP Text Encode

* 作用

将输入的 Prompt 转换为向量，支持正向描述和负向描述

* 输入

clip：来自 Load Checkpoint 的 CLIP 模型

text：提示词

* 输出

conditioning：Prompt 对应的潜在条件，是一个多维向量，用于指导生成过程

* 使用

Add Node -> conditioning -> CLIP Text Encode (Prompt)
