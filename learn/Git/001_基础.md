# Git 学习


### 安装

* Ubuntu

```bash
sudo apt-get install git
```


### 配置信息

* 发送自己的消息

```bash
git config --global user.name 'name'
git config --global user.email 'email'
# git全局配置文件在C:\Users\Administrator\.gitconfig
```

* 查看当前用户

```bash
git config user.name
git config user.email
```

* 查看配置信息

```bash
git config --list
```


### 工作区和暂存区

Git 管理的是修改，不是文件

* 未跟踪文件

新创建，没有 git add 

* 工作区

已经被跟踪，但是做了修改

* 缓存区

已经执行 git add，等待提交

* 本地

已经 commit 的，已经在本地仓库中


### 创建版本仓库（就是项目的目录）

* 创建仓库

```bash
git init	# 创建一个空目录，此时多了一个.git目录，使用ls -a可以看见
git init 目录名		# 指定目录
```


### 把修改添加到缓存区

```bash
git add 文件名
git add .		# 添加所有文件
```


### 提交

* 提交

```bash
git commit -m '说明'
```

* 查看状态

```bash
git status		# 加-s获取简短的输出
```

* 查看不同

```bash
git diff		# 未缓存的改动
git diff --cached # 已缓存的改动
git diff HEAD 	# 已缓存的和未缓存的所有改动
git diff --stat # 显示摘要
```
