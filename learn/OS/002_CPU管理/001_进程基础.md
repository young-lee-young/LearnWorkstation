# 进程


### 进程基础

* 进程

进程是对运行时程序的封装，是系统资源调度和分配的基本单位


* 进程切换

进程切换：页表 + 寄存器 + 栈


### 进程控制块 PCB（Process Control Block）

* 作用

用于描述进程


* 数据结构

```c
struct task_struct {
    pid     // 进程 ID，/proc/sys/kernel/pid_max
    ...
    *mm     // 内存资源
    *fs     // 文件系统资源
    *files  // 文件资源
    *signal
}
```


### 问题

* 取址执行，如果遇到 IO，CPU 会等待，CPU 利用率太低

CPU 交替执行多个程序，即并发
