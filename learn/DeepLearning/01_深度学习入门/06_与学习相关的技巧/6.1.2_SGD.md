# SGD（随机梯度下降法，stochastic gradient descent）


### 公式

$$
W = W - \eta \frac{\partial L}{\partial W}
$$


### 代码实现

```python
class SGD:
    def __init__(self, lr=0.01):
        self.lr = lr

    # params 保存的是权重参数
    # grads 保存的是梯度
    def update(self, params, grads):
        for key in params.keys():
            params[key] -= self.lr * grads[key]
```


### SGD 缺点

1. 容易陷入局部最优或者鞍点
2. 收敛速度慢，缺乏加速机制
3. 无法自适应调整学习率
