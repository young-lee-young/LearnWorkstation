# 数值微分


### 代码实现

```python
def numerical_diff(f, x):
    # 0.0001
    h = 1e-4
    return (f(x + h) - f(x - h)) / (2 * h)
```


### 求数值微分

```python
# 函数
def f(x):
    return 0.01 * x ** 2 + 0.1 * x

# 函数 f 真正导数的值
def f_real(x):
    return 0.02 * x + 0.1


y = numerical_diff(f, 5)
# 输出：0.1999999999990898
print(y)
r = f_real(5)
# 输出：2
print(r)

y = numerical_diff(f, 10)
# 输出：0.2999999999986347
print(y)
r = f_real(10)
# 输出：0.3
print(r)
```
