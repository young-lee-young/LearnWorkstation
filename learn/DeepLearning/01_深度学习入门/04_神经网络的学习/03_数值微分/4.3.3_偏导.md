# 偏导


### 函数

* 公式

$$
f(x_0, x_1) = x_0^2 + x_1^2
$$

* 代码实现

```python
import numpy as np


# 假如传入的时一个数组
def f(x):
    return np.sum(x ** 2)


x = np.array([1, 2])
y = f(x)
# 输出：5
print(y)
```


### 偏导数

* x1 为 4 时，求 x0 为 3 时关于 x0 的偏导

```python
def f_x(x0):
    # 这里不是偏导数，就是原函数
    return x0 ** 2 + 4 ** 2


y = numerical_diff(f_x, 3)
# 输出：6.00000000000378
# 真实的偏导数：2 * x0 = 6
print(y)
```

* x0 为 3 时，求 x1 为 4 时关于 x1 的偏导

```python
def f_y(x1):
    return 3 ** 2 + x1 ** 2


y = numerical_diff(f_y, 4)
# 输出：7.999999999999119
# 真实的偏导数：2 * x1 = 8
print(y)
```
