# 多维数组


### 多维数组

```python
import numpy as np


def multi_dimensional_array(a):
    print(a)

    # 数组的维度
    dim = np.ndim(a)
    print(dim)

    # 数组的形状，也就是每个维度有多少个元素
    # 输出的是元组
    s = a.shape
    print(s)


# 一维数组
a = np.array([1, 2, 3, 4])
multi_dimensional_array(a)

# 二维数组 
a_2 = np.array([[1, 2], [3, 4], [5, 6]])
multi_dimensional_array(a_2)
```

二维数组也叫矩阵
