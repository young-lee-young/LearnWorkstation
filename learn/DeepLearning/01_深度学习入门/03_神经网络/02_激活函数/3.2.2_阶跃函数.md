# 阶跃函数


### 作用

输入大于 0，输出 1，否则输出 0


### 公式

$$
h(x) =
\begin{cases} 
1, & x > 0 \\ 
0, & x \le 0 
\end{cases}
$$


### 代码实现

* 支持 int 阶跃函数

```python
def step_function(i):
    if i > 0:
        return 1
    return 0


x = 1
y = step_function(x)
print(y)
```


* 支持 NumPy 阶跃函数

```python
import numpy as np


def step_function(a):
    # [False, True, True]
    b = a > 0
    # return [0, 1, 1]
    return b.astype(int)


x = np.array([-1.0, 1.0, 2.0])
y = step_function(x)
print(y)
```
