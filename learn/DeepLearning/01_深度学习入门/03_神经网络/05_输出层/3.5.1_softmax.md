# softmax 函数


### 作用

1. 所有输出都在 0 - 1 之间
2. 所有输出加起来等于 1


### 公式

$$
y_k = \frac{\exp(a_k)}{\sum_{i=1}^{n} \exp(a_i)}
$$


### 代码实现

* 代码实现

```python
import numpy as np


def softmax(a):
    exp = np.exp(a)
    exp_sum = np.sum(exp)
    y = exp / exp_sum
    return y


x = np.array([0.3, 2.9, 4.0])
y = softmax(x)
print(y)
```

* 溢出

问题：上面的实现 e 的指数会非常大，导致数据溢出
解决方法：数组中每个值都减去数组中最大的值，


```python
# 溢出示例
import numpy as np


def softmax(a):
    exp = np.exp(a)
    exp_sum = np.sum(exp)
    y = exp / exp_sum
    return y


x = np.array([1010, 1000, 990])
y = softmax(x)
# 会打印出 [nan nan nan]
print(y)
```


### 防溢出代码实现

* 防溢出实现

```python
import numpy as np


def softmax(a):
    c = np.max(a)
    exp = np.exp(a - c)
    exp_sum = np.sum(exp)
    y = exp / exp_sum
    return y


x = np.array([1010, 1000, 990])
y = softmax(x)
print(y)
```
