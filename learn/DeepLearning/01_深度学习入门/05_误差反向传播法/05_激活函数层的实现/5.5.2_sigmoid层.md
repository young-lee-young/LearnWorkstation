# sigmoid 层


### 公式

$$
\frac{\partial L}{\partial x}
= \frac{\partial L}{\partial y} \cdot y(1 - y)
$$


### 代码实现

```python
import numpy as np


class Sigmoid:
    def __init__(self):
        self.out = None

    def forward(self, x):
        out = 1 / (1 + np.exp(-x))
        self.out = out
        return out

    def backward(self, dout):
        dx = dout * self.out * (1 - self.out)
        return dx
```
