# ReLU 层实现


### 公式

$$
\frac{\partial y}{\partial x} =
\begin{cases}
1 & (x > 0) \\
0 & (x \le 0)
\end{cases}
$$


### 代码实现

```python
class ReLU:
    def __init__(self):
        self.mask = None

    def forward(self, x):
        # 把 <= 0 的位置置成 True
        self.mask = (x <= 0)
        out = x.copy()
        # 把 <= 0 的地方设置成 0
        out[self.mask] = 0
        return out

    def backward(self, dout):
        # > 0: ∂y/∂x
        # <= 0: 0
        dout[self.mask] = 0
        dx = dout
        return dx
```
