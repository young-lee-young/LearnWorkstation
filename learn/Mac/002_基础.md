# mac使用


### 终端自动补全的配置

* 打开终端，输入

```bash
vim .inputrc
```

* 在文件里面写上：

```bash
set completion-ignore-case on
set show-all-if-ambiguous on
TAB: menu-complete
```

ctrl + o ,回车，重启终端，自动补全按tap键就ok。


### 显示或隐藏隐藏的文件

```sh
shift + command + .
```


### 文件剪切

```bash
command + C	复制文件
command + option + V	将图片移动到指定目录
```


### 开启多个终端窗口

```sh
# 新建一个窗口
command + N

# 新建一个tab
command + T
```


### 终端快捷键

```sh
# 删除一行
control + u
# 删除前一个单词
control + w

# 移动到行首
control + a
# 移动到行尾
control + e
```


### 编辑器

```bash
# 行首或者行尾
command + 左/右

# 按单词跳转光标
option + 左/右

# 删除单词
command + 回车/DELETE

# 删除整行
command + shift + 左/右
```


### 谷歌浏览器开启开发者模式

```sh
option + command + i/c
```
