# RIP 路由协议


### 简介

* 距离矢量路由协议
* 传闻更新（不知道整个网络拓扑结构）


### RIP路由协议

根据跳数选择路由，30秒广播路由表，最大跳数16跳（大于16跳，认为目标主机不可到达）

* 查看路由器运行的协议

```bash
show ip protocols
```


* 配置RIP协议

```bash
# 全局模式					

# 启用RIP协议
router rip

# 运行版本1
version 1

# 指定参与动态路由的网段
network 网段

# 删除参与动态路由的网段
no network 网段
```


* 查看RIP更新活动

```bash
# 开启RIP debug
debug ip rip

# 关闭RIP debug
undebug all
```


* 删除RIP路由协议

```bash
# 全局模式
no router ip
```
