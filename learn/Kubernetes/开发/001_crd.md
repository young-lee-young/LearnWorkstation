# KubeBuilder

KubeBuilder 文档：https://book.kubebuilder.io


### 项目搭建

* 初始化项目

```bash
# 创建项目
mkdir helloworld

# kubebuilder init --domain $domain --repo $repo
# domain: group 的域名
# repo：go.mod 中的 module
kubebuilder init --domain com.lee --repo github.com/lee/helloworld
```


* 创建 API

```bash
# kubebuilder create api --group $group --version $version --kind $kind
# group：组
# kind：一定是大写开头
kubebuilder create api --group lee --version v1beta1 --kind Book
```


* 编辑 API 字段

在 api/$version/$kind_types.go 添加需要的字段

```go
package v1beta1

import	metav1 "k8s.io/apimachinery/pkg/apis/meta/v1"

type xxxSpec struct {
	xxx string `json:"xxx,omitempty"`
}

type xxxStatus struct {
}

type xxx struct{
	metav1.TypeMeta   `json:",inline"` // 保存 Version、Kind 信息
	metav1.ObjectMeta `json:"metadata,omitempty"` // 保存资源对象名称、namespace 信息

	Spec   BookSpec   `json:"spec,omitempty"` // 期望状态，specification
	Status BookStatus `json:"status,omitempty"` // 真实状态
}
```


* 生成 CRD

```bash
# 会在 config/crd/bases/ 生成 yaml 文件
make manifests
```


* 安装 CRD

```bash
make install
```


* 卸载 CRD

```bash
make uninstall
```
