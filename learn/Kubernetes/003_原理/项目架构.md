基于 Kubernetes v1.24


### 项目结构

api 目录：定义 openAPI

cmd 目录：模块入口

hack 目录：工程编译、测试执行、E2E 环境准备等命令行操作

pkg 目录：主要逻辑目录

staging 目录：准备独立出去的子模块

vendor 目录：项目外部依赖


### staging/src 目录

apimachinery: 原始类型

api：内置 API 的外部类型定义

apiserver: Generic Server

kube-aggregator: 聚合器

code-generator: 代码生成


### pkg 目录

api: 历史遗留产物

apis: 内置 API 的内部类型定义

controlplane: 核心 API Server 代码

kubeapiserver: Master Server 代码（主 Server）

registry: 和 etcd 交互的代码，HTTP Server 响应
