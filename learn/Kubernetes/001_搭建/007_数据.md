* 查看所有的 key

```bash
etcdctl get / --prefix --keys-only
```


* 查看某个数据，并以 json 格式显示

```bash
etcdctl get $key -w=json | python3 -m json.tool
```


### minikube

```bash
# 进入到 etcd 容器中
etcdctl --endpoints=https://127.0.0.1:2379 --cacert=/var/lib/minikube/certs/etcd/ca.crt --cert=/var/lib/minikube/certs/etcd/server.crt --key=/var/lib/minikube/certs/etcd/server.key get /registry/pods/kube-system/etcd-minikube
```
