# 调试 Kubernetes


### 调试 kube-apiserver

```bash
# 查看 kube-apiserver 进程，复制整个进程启动参数
ps -ef | grep kube-apiserver

# kill 掉 kube-apiserver 进程

# 使用 dlv 启动 kube-apiserver
sudo dlv --listen=:$PORT --headless exec $kube-apiserver_path --api-version=2 --log --log-output=debugger,gdbwire,lldbout,debuglineerr,rpc,dap,fncall,minidump --log-dest=$log_path -- $进程启动参数
```
