### 项目启动

* IDEA启动

* 命令行启动

```sh
# 进入到项目目录下
mvn spring-boot:run
```

* 打包启动

```sh
# maven清除包
mvn clean

# 打包跳过单元测试，-f指定pom文件
mvn package -Dmaven.test.skip=true -f pom.xml

# 在target文件夹下会生成一个jar文件，使用java命令运行
java -jar jar文件

# 指定配置文件启动
java -jar -Dspring.profiles.active=pro jar文件
```
