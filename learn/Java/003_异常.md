# Java异常


### 异常类父类Throwable

有Error类和Exception两个子类


### Exception类

1. RuntimeException（运行时异常，非检查异常）

空指针异常，数组下标越界异常，类型转换异常，算术异常

2. 检查异常

文件异常，连接异常
