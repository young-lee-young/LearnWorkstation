### 变量定义

* 使用 let 定义变量，定义的变量不可变，定义变量并赋值的过程称为'变量绑定'

```rust
fn main() {
    let a = "lee";

    // 这样是错误的，因为变量不可变（immutable）
    a = "zhao";
    
    println!("{}", a);
}
```


* 使用 mut 定义可变变量

```rust
fn main() {
    let mut a = "lee";
    
    // 这样是可以的，mut（mutable）定义的变量是可变的
    a = "zhao";
    
    println!("{}", a);
}
```


### 忽略未使用的变量

```rust
fn main() {
    a = "lee";
    
    // 如果变量未使用，编译会有警告，使用下划线_开头忽略未使用的变量
    _b = "zhao";
    
    println!("{}", a);
}
```


### 常量

常量使用下划线 _ 分割单词

```rust
// 定义字符串常量
const MY_NAME: &'static str = "zhao";

// 定义数字常量，数字可以通过 _ 分割，方便阅读
const AGE: u64 = 100_0000;

fn main() {
    println!("{}", MY_NAME);
    println!("{}", AGE);
}
```

* 🤔️：不可变变量和常量的区别是什么？

常量在编译期就确定了，不允许运行时计算，而不可变变量是在运行时才知道值


### 变量遮蔽

作用：在局部作用域内，可以覆盖外层的变量

```rust
fn main() {
    let x = 5;
    // 生成了完全不同的新变量，只是名称相同，涉及内存对象的再分配
    // 作用：如果不需要再使用之前的变量，可以重复使用这个变量名，不需要想更多的变量名
    let x = x + 1;
    {
        let x = x * 2;
        // 输出：12
        println!("x in is {}", x);
    }
    // 输出：6
    println!("x is {}", x);
}
```
