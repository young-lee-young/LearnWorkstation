# 类型

### 数值类型

* 类型

有符号整型 i8、i16、i32、i64、isize（i 表示 integer）
无符号整型 u8、u16、u32、u64、usize（u 表示 unsigned）

默认为：i32

```rust
fn main() {
    let a: i64 = -1;

    let b: u64 = 2;

    println!("a is {}, b is {}", a, b);
}
```

* 整数溢出

```rust
fn main() {
    let a: u8 = 255;

    // 发生溢出时，返回溢出后的值
    let ret: u8 = a.wrapping_add(1);
    println!("{}", ret); // 输出：0
}
```

```rust
fn main() {
    let a: u8 = 255;
    
    // 发生溢出时，返回 None
    let ret = a.checked_add(1);
    println!("{:?}", ret) // 输出：None
}
```

```rust
fn main() {
    let a: u8 = 255;

    // 发生溢出时，返回溢出后的值和是否溢出标志
    let (ret, overflow): (u8, bool) = a.overflowing_add(1);
    println!("{}, {}", ret, overflow); // 输出：0, true
}
```

```rust
fn main() {
    let a: u8 = 255;

    // 该类型的最大值
    let ret: u8 = a.saturating_add(1);
    println!("{}", ret); // 输出：255
}
```

* 浮点数缺陷

避免陷阱，需要遵守准则：

1. 避免在浮点数上测试相等性
2. 当结果在数学上可能存在未定义时，需要格外小心

```rust
fn main() {
    // 此断言会导致 panic
    assert!(0.1 + 0.2 == 0.3);

    // 浮点数运算做法
    assert!((0.1_f64 + 0.2 - 0.3).abs() < 0.00001)
}
```

```rust
fn main() {
    // x 为 NaN
    let x = (-42.0_f32).sqrt();
    println!("x is nan: {}", x.is_nan());
}
```

### 序列

```rust
fn main() {
    // 0 < i < 5
    for i in 1..5 {
        println!("{}", i);
    }

    // 0 < i <= 5
    for i in 1..=5 {
        println!("{}", i);
    }

    // 'a' < i <= 'z'
    for i in 'a'..='z' {
        println!("{}", i);
    }
}
```
