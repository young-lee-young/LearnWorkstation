### 变量定义

* 使用 let 定义变量，定义的变量不可变

```rust
fn main() {
    let a = "lee";

    // 这样是错误的，因为变量不可变（immutable）
    a = "zhao";
    
    println!("{}", a);
}
```


* 使用 mut 定义可变变量

```rust
fn main() {
    let mut a = "lee";
    
    // 这样是可以的，mut（mutable）定义的变量是可变的
    a = "zhao";
    
    println!("{}", a);
}
```


### 忽略未使用的变量

```rust
fn main() {
    a = "lee";
    
    // 如果变量未使用，编译会有警告，使用下划线_开头忽略未使用的变量
    _b = "zhao";
    
    println!("{}", a);
}
```
