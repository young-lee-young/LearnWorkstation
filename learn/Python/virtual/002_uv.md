### uv 基本操作

* 安装 uv

```bash
pip install uv
```


* 管理 python 版本

```bash
# 会下载到 /root/.local/share/uv/python 下
uv python install 3.10
```


* 创建虚拟环境

```bash
# uv venv $env_name --python=$version
uv venv lee --python=3.10
```


* 激活虚拟环境

```bash
# source $env_name/bin/activate
source lee/bin/activate

# 安装依赖包
uv pip install $package

# 查看安装的 python 依赖包
uv pip list

# 退出虚拟环境
deactivate
```
