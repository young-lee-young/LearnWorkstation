# 面板管理


* 将面板分为左右两块

```bash
tmux split-window -h
# 快捷键：同时按 ctrl + b，再按 %
```

* 将面板分为上下两块

```bash
tmux split-window
# 快捷键：同时按 ctrl + b，再按 "
```

* 切换面板

```bash
# 切换到上方面板
tmux select-pane -U

# 切换到下方面板
tmux select-pane -D

# 切换到左方面板
tmux select-pane -L

# 切换到右放面板
tmux select-pane -R
```

快捷键：同时按 ctrl + b，再按 上、下、左、右 方向键
