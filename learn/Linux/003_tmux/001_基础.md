# tmux基础


### 安装

* Mac 安装

```bash
brew install tmux
```


### 启动和退出

```bash
# 启动 tmux
tmux

# 退出tmux
ctrl + d
```


### 会话管理

```bash
# 新建会话
tmux new -s $session_name

# 查看会话
tmux ls

# 分离会话
tmux detach

# 接入会话
tmux attach -t $session_name

# 杀死会话
tmux kill-session -t $session_name

# 切换会话
tmux switch -t $session_name

# 重命名会话
tmux rename -t $old_name $new_name
```
