# 过程（子程序）调用指令

字程序：程序中具有独立功能的部分编写成独立程序模块



### 子程序调用、返回指令：call、ret

将 IP 寄存器中的值入栈，然后 jmp 到标签处

ret：pop ip

```assembly
call print

print:
		...
		ret
```



### 远调用：call far、retf

将 CS、IP 寄存器中的值入栈，然后 jmp 到指定段的标签处

retf：pop ip && pop cs

```assembly
call far [print] ; 也可以写为 call 0:print

print:
	...
	retf
```
