# 循环控制指令



### 无条件循环：loop

遇到 loop 指令，首先执行 cx = cx - 1，然后判断 cx 的值，若 cx != 0，则转移到 Lable 处执行，否则继续向下执行

```assembly
; 求 1 + 2 + 3 + ... + 99 + 100 值

mov ax, 0
mov cx, 100

sum:
		add ax, cx
		loop sum
```
