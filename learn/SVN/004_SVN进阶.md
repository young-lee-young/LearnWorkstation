# SVN进阶学习


### 客户端进阶命令

```sh
svn list            # 列出所有处于版本控制的文件

svn status          # 列出工作副本文件夹状态
# ?,无版本控制
# D,从版本库删除
# M,被编辑
# A,被添加到版本库中
# R,文件被替换(删除后创建同名的文件)
# C,文件存在冲突
# !,文件缺失

svn log             # 提交日志信息

svn info            # 工作副本及文件的详细信息
svn info --xml

# 复制并加入到版本控制
svn copy 文件名 目标文件名
svn copy -r 4 文件名 目标文件名
# 批量复制
svn copy 文件名1 文件名2 目标路径
```
