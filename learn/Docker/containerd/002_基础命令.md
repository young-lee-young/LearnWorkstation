# containerd


### namespace

```bash
sudo ctr namespace list

sudo ctr ns list
```


### 镜像

* 拉取镜像

```bash
ctr --namespace=$namespace images pull docker.io/library/ubuntu:20.04
```

* 查看镜像

```bash
ctr --namespace=$namespace images list
```

* 删除镜像

```bash
ctr --namespace=$namespace images remove $image_name
```

* 导入镜像

```bash
ctr --namespace=$namespace images import alpine:lee.tar --platform linux/amd64
```


### 容器

* 启动容器

```bash
ctr --namespace=$namespace run $image $container_name
```

* 查看容器

```bash
ctr --namespace=$namespace container list
```

* 进入容器

```bash
ctr --namespace=$namespace task exec -t --exec-id $exec-id $container /bin/sh
```
