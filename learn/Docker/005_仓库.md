# 仓库



### 安裝

```sh
# 拉取registry镜像
docker pull registry

# 运行容器
docker run -d -p 5000:5000 -v 本地映射地址:/var/lib/registry

# 打开浏览器，访问http://地址:5000/v2/_catalog
```



### 上传到镜像仓库

```bash
# 登录docker
docker login
# 如果登录报：error storing credentials - err: exec: "docker-credential-desktop": executable file not found in $PATH，修改"~/.docker/config.json"，将"credsStore" : "desktop"删除

# 镜像打TAG，否则推送不上去，镜像名:版本 用户名/镜像名:版本
docker tag go-gin:v1 1135479869/go-gin:1.0

# 推送
docker push 1135479869/go-gin:1.0
```
