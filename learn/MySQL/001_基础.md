# MySQL的操作


### 安装

* Windows安装

1. MSI安装

2. 启动MySQL

```sh
net start 服务名
net stop 服务名
net restart 服务名
```

* Mac安装

1. HomeBrew安装

```sh
# 安装MySQL
brew install mysql

# 启动MySQL
brew services start mysql

# 关闭服务
mysql.server stop
```

* Linux安装

```sh
sudo apt-get install mysql-server mysql-client
# 进入/etc/mysql/mysql.conf.d，编辑mysqld.cnf文件
```

* 重启服务

```sh
sudo service mysql restart
```


### 登陆与登出

* 登录

```sh
mysql -u root -p
```

* 登出

```sh
exit;
```
