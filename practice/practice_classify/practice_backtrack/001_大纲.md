### 回溯算法

* 回溯使用在哪里

通常是递归函数的下面


* 效率

纯暴力搜索，并不是高效的算法


* 回溯解决的问题

1. 组合问题（不强调元素的顺序）
2. 切割问题
3. 子集问题
4. 排列问题（强调元素的顺序）
5. 棋盘问题（N 皇后）


* 回溯算法

1. 递归函数的参数和返回值
2. 递归函数结束条件
3. 单层搜索逻辑


* 回溯算法模版

```go
package backtrack

func backtrack(参数) {
	if 递归终止条件 {
		收集结果
		return
	}
	
	// 单层搜索逻辑
	for 集合的元素集 {
		处理节点
		
		递归函数
		
		回溯操作
	}
	
	return
}
```
