### 优先队列

按照元素优先级取出，优先级是自己定义的一个元素属性

实现：二叉堆、二叉平衡树

* 时间复杂度

访问最值：O(1)
插入：一般是 O(logn)，一些高级数据结构（如斐波那契堆）可以做到O(1)
取最值：O(logn)

### 表达式求值

* 前缀表达值（波兰式）

op A B
例：* 3 + 1 2 = * 3 3

* 后缀表达式（逆波兰式）

A B op
例：1 2 + 3 * = 3 3 *

* 中缀表达式

A op B
例：3 * (1 + 2) = 3 * 3
