# 查找表

* set 和 map 底层实现

|     | 普通数组  | 顺序数组 ｜ 二分搜索树
|  ----  | ----  | ---- ｜ ----
| 插入  | O(1) | O(n)（元素需要后移） | O(logn)
| 查找  | O(n) | O(logn) ｜ O(logn)
| 删除  | O(n) | O(n)（元素需要移动） ｜ O(logn)

* 哈希表相对二分搜索树的缺点

失去了数据的顺序性
