# 数组


### 题型分类

* 快慢指针
* 三路快排
* 相向指针
* 滑动窗口


### 题目整理

* 快慢指针

✅ 001 将所有0移动到数组末尾（例题）LeetCode No283
✅ 002 删除数组中指定值 LeetCode No27
✅ 003 删除数组中重复值 LeetCode No26
✅ 004 删除数组中重复值，每个数最多保留2个 LeetCode No80

✅ 005 三种颜色排序（例题）LeetCode No75

* 相向指针

✅ 008 有序数组两数之和（例题）LeetCode No167
